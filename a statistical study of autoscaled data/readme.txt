Data and files in this folder are for publish of A statistical study of autoscaled data at the northern equatorial ionization anomaly during the year 2016. 
The file "autoscaled.zip" includes *.para files which are autoscaled parameters from ionograms by ionoScaler.
The file "manualscaled.zip" includes *.para files which are manualscaled parameters from ionograms by the operator.
The file "ReadIonoPara.m" can read autoscaled and manual data.
