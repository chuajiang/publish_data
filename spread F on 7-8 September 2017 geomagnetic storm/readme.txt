Data and files in this folder are for publish of  Ionosonde observations of spread F/ionospheric irregularities at low and middle latitudes during the 7-8 September 2017 geomagnetic storm
The files "*.rar" includes *.para files which are autoscaled parameters from ionograms by ionoScaler.
The file "cnroti_20170908_20170912" is assocaited with GNSS observations.
The file "ReadIonoPara.m" can read ionogram data.
